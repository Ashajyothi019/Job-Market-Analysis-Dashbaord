📊 Job Market Analysis Dashboard (Excel)
🔍 Overview

This project analyzes a Kaggle job dataset to explore salary trends, job roles, experience levels, and skill demand in the data analytics domain. An interactive Excel dashboard was built for insights.

🎯 Objectives
Analyze salary by role and experience
Identify in-demand skills
Compare remote vs onsite jobs
Build an interactive dashboard
🛠️ Tools Used
Excel
XLOOKUP, INDEX-MATCH
IF, AVERAGEIF, COUNTIF
Pivot Tables & Charts
📈 Key Insights
AI/ML roles have highest salaries
Salary increases with experience
Python is the most demanded skill
Many jobs are remote
